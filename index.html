Keygen

Pattern-based key generator

The Keygen plugin is a key or serial generator based on user-defined mask patterns. The plugin is able to generate keys like the example below:

6W2F-4RJB-KV0Z-ADA6-4SJ2

It can be used to generate keys for several purposes, some examples are:

  • Unique links to reset passwords
  • Registration via invitation link
  • Two-Factor authentication codes
  • File sharing links
  • Product redemption keys
  • Discount coupons
  • Referral codes

But why a Non-Styled website?

We are very used to modern designs like Material Design and Bootstrap for example.
I decided to write my website completely without CSS in an attempt to make it possible to convey the necessary information to the user, without the content necessarily being beautiful or visually pleasing.
We don't need beautiful websites, we need websites with decent content.
For future personal projects, I intend to use the same idea on even bigger things.
